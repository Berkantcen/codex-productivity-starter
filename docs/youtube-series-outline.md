# YouTube Series Outline

This repository can support a practical video series about working with Codex.

## Episode 1 - Make Any Repo Codex-Ready

Show how to copy the starter files into an existing project and fill the first
usable version.

Repo files:

- `templates/project-starter/AGENTS.md`
- `templates/project-starter/repo-map.md`
- `templates/project-starter/validation-plan.md`

## Episode 2 - Stop Repeating Context

Show how `repo-map.md`, `decision-log.md`, and scoped task briefs reduce repeated
prompt text.

Repo files:

- `docs/token-efficiency.md`
- `templates/project-starter/decision-log.md`
- `templates/project-starter/task-brief.md`

## Episode 3 - Better Handoffs And Validation

Show the difference between a vague final answer and a useful engineering
handoff with command results, risks, and review order.

Repo files:

- `templates/project-starter/handoff-summary.md`
- `templates/project-starter/validation-plan.md`

## Episode 4 - Real Bug Investigation Workflow

Use the bug investigation example to demonstrate a narrow investigation before
coding.

Repo files:

- `examples/bug-investigation/task-brief.md`

## Episode 5 - From Template To Plugin

Explain when a simple repo template is enough and when it is worth packaging the
workflow as a Codex plugin.

Repo files:

- `README.md`
- `docs/adoption-playbook.md`
